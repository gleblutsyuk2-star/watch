Какие скиллы я бы использовал

Насколько я помню наш набор скиллов для Antigravity, вот что обязательно стоит подключить.

⭐ Обязательные
1. Taste Skill ⭐⭐⭐⭐⭐

Это вообще №1.

Он будет следить за:

композицией;
типографикой;
правильными отступами;
общей эстетикой.

Без него такой проект я бы даже не начинал.

2. GSAP Skill ⭐⭐⭐⭐⭐

Это будет половина проекта.

Через GSAP можно сделать:

плавное вращение часов;
раскрытие механизма;
parallax;
плавные переходы;
анимацию цифр;
cinematic scroll.
3. Motion Skill ⭐⭐⭐⭐⭐

Чтобы агент не делал обычные fade-in.

Нужны дорогие motion-переходы.

Например:

появление текста

↓

появление света

↓

вращение часов

↓

смена секции.

4. UI/UX Skill ⭐⭐⭐⭐⭐

Чтобы блоки были построены профессионально.

5. Accessibility Skill ⭐⭐⭐⭐☆

Даже для luxury проекта.

Особенно контрастность и читаемость.

Желательно
Three.js Skill ⭐⭐⭐⭐⭐

Если он у тебя есть — вообще идеально.

Можно сделать:

вращение часов мышкой;
отражение света;
стекло;
металл;
золотые элементы.

Это сразу поднимет проект на другой уровень.

Scroll Animation Skill ⭐⭐⭐⭐⭐

Для всей логики скролла.

если не находишь скилл у меня на компьютере , то можешь сам его скачать через терминал в проект.
